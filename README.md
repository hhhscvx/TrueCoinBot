# TrueCoin
TrueCoin claimer https://t.me/true_coin_bot

### Configuration ⚙️
1. Get api id and api hash from [here](https://my.telegram.org/auth "here")
2. Create `.env` and put your api id and api hash like:
```
API_ID=your_api_id
API_HASH=your_api_hash
API_KEY=your_api_key
```
### Get API Key:
1. Repeat the step from that video: https://youtu.be/NYxHmck_GjE
2. Launch mini app -> right click -> inspect element
3. Go to network -> Click on any request -> and get Auth-Key from Request Headers

### Library Installation 📚
1. `python3 -m venv venv`
2. `source venv/bin/activate` for linux
3. `.\venv\Scripts\activate.bat` for Windows
4. `poetry init`
5. `poetry install`

### Start bot 🤖:
`python3 main.py`

### Python Version 🐍
Preferably python 3.10+
